# 闹钟音频文件目录

这个目录用于存放闹钟提醒的音频文件。

## 推荐的音频格式
- MP3 格式
- 文件大小建议小于 5MB
- 音频长度建议 10-30 秒

## 默认音频文件
系统会尝试使用以下音频文件作为默认闹钟铃声：
- `alarm.mp3` - 主要闹钟音频
- `notification.mp3` - 备用通知音频

## 添加自定义音频
1. 将您的音频文件复制到此目录
2. 确保文件名为英文，不包含特殊字符
3. 在应用的闹钟设置中选择您的自定义音频

## 注意事项
- 如果音频文件不存在，系统将使用默认的系统提示音
- 建议使用较为温和的音频作为闹钟，避免过于刺激的声音
